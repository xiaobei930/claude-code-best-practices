8 Specialized Agents

PM, Lead, Designer, Dev, QA, Architect, Code Reviewer, and more. Each agent handles its domain autonomously.

43 Commands

Complete CLI coverage. From /iterate for autonomous loops to /security-audit, /confidence-check, /commit, /verify, /release and everything in between.

Two Modes

/iterate for fully autonomous development loops. /pair for step-by-step collaborative programming.

18 Skills + 33 Rules

Industry best practices for frontend, backend, database, DevOps, security, testing, and more — built into every interaction.

21 Safety Hooks

Pre and post tool-use hooks that prevent dangerous operations, enforce coding standards, auto-format code, and run config security audits.

Memory Bank

Persistent project memory across sessions. Architecture decisions, tech stack, progress — all remembered and applied.
